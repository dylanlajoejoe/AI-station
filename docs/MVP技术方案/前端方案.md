# 前端方案

## 1. 技术栈

- React
- TypeScript
- Vite

## 2. 负责内容

- 工作区页面
- 文件树展示
- 文件夹展开和折叠
- 文件或文件夹右键菜单
- `添加到对话` 插入输入框
- 对话输入框
- 对话消息展示
- 历史会话列表展示

## 3. 核心交互

1. 用户选择目录
2. 前端请求本地服务获取目录树
3. 用户右键文件或文件夹
4. 用户点击 `添加到对话`
5. 前端把文件或文件夹全名插入输入框
6. 用户发送消息
7. 前端展示 AI 回复

## 4. 不做

- 不读取文件内容
- 不上传文件
- 不解析文件
- 不维护文件上下文
